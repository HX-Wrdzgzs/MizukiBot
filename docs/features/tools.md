# 通用工具与娱乐

> 包含表情包系统、Minecraft、Steam 及 AI 语音。

<br>

<div class="custom-card" style="background: var(--vp-c-bg-soft); border: 2px solid #b37feb; padding: 20px; border-radius: 15px; margin-bottom: 25px;">
  <h3 style="margin-top: 0; color: #722ed1;">表情包系统 (Meme)</h3>

  <h4 style="color: #722ed1; margin-top: 15px;">基础指令</h4>

| 指令 | 说明 |
| :--- | :--- |
| `表情帮助` | 查看帮助 |
| `发送 [文件夹名]` | 随机发一张 (例: `发送 mzk`) |
| `查看 [图片名]` | 精确查看 (例: `查看 mzk1`) |
| `看所有 [文件夹]` | 生成预览图 |

  <h4 style="color: #722ed1; margin-top: 25px;">上传与管理</h4>

| 指令 | 说明 |
| :--- | :--- |
| `[文件夹] 上传 [图]` | 上传图片到本地库 |
| `[文件夹] pic上传 [图]` | 上传到 API 库 |
| `最近上传` | (管理) 查看入库记录 |
| `删除 [图片名]` | (管理) 删除图片 |
| `溯源 [图片名]` | (管理) 查看上传者 |

</div>

| `今日DAU` | (管理) 查看日活人数 |

</div>

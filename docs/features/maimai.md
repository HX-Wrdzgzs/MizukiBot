# 舞萌 DX 专项

> 查分、查歌与排队服务。

<br>

## 🟡 核心查询

| 指令 | 说明 |
| :--- | :--- |
| `今日舞萌` | 运势与推荐 |
| `b50` | 查B50成绩 |
| `随个 [难度]` | 随机歌 (例: `随个紫`) |
| `查歌 [关键词]` | 查曲目信息 |
| `定数查歌 [范围]` | 例: `定数查歌 13.5 13.9` |

<br>

## 🟢 牌子与进度

| 指令 | 说明 |
| :--- | :--- |
| `[牌子]进度` | 例: `霸者进度` |
| `[等级]进度` | 例: `13+进度` |
| `完成表 [等级]` | 生成完成度表格 |
| `分数线 [曲ID]` | 查分数线详情 |

<br>

## 🔵 别名与娱乐

| 指令 | 说明 |
| :--- | :--- |
| `是什么歌` | 查别名对应曲 |
| `添加别名 [ID] [名]` | 添加新别名 |
| `mai猜歌` | 开启猜歌游戏 |

<br>

## 🟠 骚话模式 (娱乐)

> 仅供娱乐，切勿当真

* “xxmai什么” (例: `柚子mai什么`)
* “我要在 [难度] 上 [分数] 分”
* “今天mai出勤打什么上分”
* “[等级] [评价] 进度” (例: `13+ SSS 进度`)

<br>

## 🖥️ 舞萌服务器状态
> 获取 Milk/Chongxi/DLX/华立 代理服务器实时状态。

* 指令: `mai状态` / `看看dlx` / `看看华立`
* [👉 点击访问 status.mizuki.top 查看网页版状态](https://status.mizuki.top)

<br>

## 🔄 成绩同步助手 (Sync)

> 协助将街机端游玩记录同步至查分器 (水鱼/落雪)。
> **⚠️ 维护时间: 每日 00:00-11:00 暂停服务**

### 1. 设置与绑定
* `设置类型 1`：仅上传成绩
* `设置类型 2`：上传后自动登出 (传占位曲)
* `绑定水鱼 [Token]`
* `绑定落雪`

### 2. 同步与查询
* `上传成绩`：发送后发二维码
* `预览mai`：查询游戏账号基本信息

### 3. 队列与管理
* `队列状态` / `登出` / `查看绑定`

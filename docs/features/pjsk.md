# 🎵 PJSK 相关功能 (Project Sekai)

> 包含查分、查榜、组卡计算器、活动预测及各类娱乐功能。
> <br><small>*部分功能支持 `cn` (国服) 或 `tw` (台服) 前缀。*</small>

<br>

## 📊 玩家数据查询 (需上传数据)

> 以下功能需要先上传数据。<br>
> **日/台服上传地址：** [点击进入数据上传页面](http://go.mikuware.top/)<br>
> <small>(日服玩家可直接复制引继码给Bot上传)</small>

* `rk`：获取玩家排位信息
* `b39`：Rating最高39首 (仅供参考)
* `pjsk b30`：Rating最高30首 (算法不同)
* `pjsk进度 [ex/apd]`：查Master/Expert完成表
* `pjskdetail`：查金币、水晶、练度 (建议私聊)
* `视奸`：查看当前队伍综合力构成
* `逮捕`：查询高难谱面 AP/FC 进度
* `pjskcard`：个人卡牌图鉴 (可加角色花名)
* `pjskevent`：查询前10次最高排名活动
* `最强组卡`：同色同队最高分卡组

<br>

## 🎵 查歌与活动 (无需上传)

### 基础功能
* `绑定 [ID]`：绑定游戏ID
* `给看/不给看`：开放/关闭被At权限
* `pjskprofile`：获取旧版个人信息卡片
    * `查时间`：查询注册时间
    * `id`：返回当前QQ绑定ID

### 活动与榜单
* `sk`：查活动排名/档线
* `分数线/sk线`：查当前档线分数 (5分延迟)
* `ss/wlss`：查当前/单榜档线时速
* `sk预测`：(日服) 查预测线
* `查房`：查最近1小时活动情况
* `活动 [编号]` / `findevent`：查活动信息

### 歌曲查询
* `pinfo [曲名]` (查详情) / `查bpm` / `查物量`
* `谱面预览 [曲名] [ex/apd]` / `技能预览`
* `难度排行 [level] [难度]` (例: 难度排行 33 master)

<br>

## 🧮 组卡计算器

| 指令 | 说明 |
| :--- | :--- |
| `组卡 [队] [色]` | **指定箱活最佳卡组**<br>例: `组卡 ln 橙`<br>队名: ln/mmj/vbs/ws/25/vs |
| `活动组卡 [曲]` | **当前活动最佳卡组**<br>例: `活动组卡 独瑞 hard`<br>默认歌曲: 独りんぼエンヴィー |
| `挑战组卡 [人]` | **每日挑战Live最佳卡组**<br>例: `挑战组卡 miku` |

<br>

## 🃏 卡面查询

> **🔎 查卡 (findcard) 详细说明**
> * 1个参数: 角色名/队名 (例: `miku`, `25`)
> * 2个参数: +属性/稀有度 (例: `miku 4`, `25 紫`)
> * 3个参数: +属性+稀有度 (例: `miku 4 限定`)

* `card [编号]` (查大图) / `cardinfo [编号]` (查数值)
* `看 [角色]` (随机卡图)

<br>
